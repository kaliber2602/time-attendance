I. Setup instructions:

1. Install Nodejs (If you have not installed)
https://nodejs.org/en/download

2. Download the source code from GitHub and extract the ".zip" file
https://github.com/kaliber2602/time-attendance.git

3. Go to the project folder and install required dependencies (Using terminal):
npm install

5. To run the code:
npm start

Project will be automatically open at http://localhost:3000
